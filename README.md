# Midterm_Project_ML
Includes model development notebook, trained model (pickle), Streamlit app, and deployment files.

**Dataset source** : https://www.kaggle.com/datasets/muhammadshahidazeem/customer-churn-dataset 

<br>
## Project Overview
Customer churn is when a customer stops using a service. This project predicts whether a customer is likely to **churn (leave)** or **stay** based on behavioral, subscription, and billing data.  

<br> 
## Files in the Repository
| File/Folder                  | Description |
| `churn_model.pkl`            | pkl fine that includes trained model, scaler, encoder. |
| `01_churn.ipynb`             | initial Notebook for data preprocessing, model training, and evaluation. |
| `churn_streamlit.py`         | Streamlit app source code for predicting customer churn. |
| `Images/`                    | Contains logos used in the Streamlit app. |
| `requirements.txt`           | python library dependency |
| `raw_data`                   | datasets |
